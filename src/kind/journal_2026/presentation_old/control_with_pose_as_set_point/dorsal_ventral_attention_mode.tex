\begin{frame}{Dorsal and ventral attention network mode}
    Two large-scale brain network which require attention unlike default mode network
    \begin{itemize}
        \item Ventral network mode (Activates for a new observation or discrepency) \cite{scalf-2014-trial-history-effects-in-the-ventral-attentional-network}
        \item Dorsal network mode (Activates when brain tries to follow a goal such spatial goal) \cite{vossel-2014-dorsal-and-ventral-attention-systems, corbetta-2002-control-of-goal-directed-and-stimulus-driven-attention-in-the-brain}
    \end{itemize}
\end{frame}
