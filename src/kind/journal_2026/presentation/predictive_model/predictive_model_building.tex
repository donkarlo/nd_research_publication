\begin{frame}{Predictive brain}
    \cite{spratling-2017-a-review-of-predictive-coding-algorithms} introduces the shared idea that a brain uses a predictive model to explain sensory input by reducing prediction error.
    
    \textit{Active inference} \cite{friston-2005-a-predictive-brain-how-perception-and-action-are-coupled-in-neural-networks} is one among the many brain theories that considers brain as predictive machine


\end{frame}
