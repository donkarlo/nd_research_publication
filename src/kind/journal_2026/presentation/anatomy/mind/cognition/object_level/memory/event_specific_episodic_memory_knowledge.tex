\begin{frame}[allowframebreaks]{Event specific episodic memory knowledge}
    \begin{itemize}
        \item \cite{brewer-1986-what-is-autobiographical-memory} defines as memory for information related to the self
        \item Will be saved and remembered as a language
        \item The autobiographical knowledge base contains knowledge of the self, used to provide information on what the self is, what the self was, and what the self can be.
    \end{itemize}
    
    
    \textbf{Event specific Knowledge}
        \cite{pillemer-2001-momentous-events-and-the-life-story} describes it as a vividly detailed information about individual events, often in the form of visual images and sensory-perceptual features.

\end{frame}
