\begin{frame}[allowframebreaks]{Language}
    Language is a symbolic system that, in its most basic form, consists of sequences of replaceable components used to convey different meanings \cite{saussure-1916-course-in-general-linguistics}
    So to build a language we need:
    \begin{itemize}
        \item Reusable parts
        \item A sequence to sequence model which predict next utterance according to previous utterences
    \end{itemize}
\end{frame}
