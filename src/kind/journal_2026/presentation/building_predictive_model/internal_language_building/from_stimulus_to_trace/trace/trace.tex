\begin{frame}[allowframebreaks]{Trace}
    A trace ( in linguistic called \alert{feature structure}) is an attribute--value description of a linguistic object, recursively built from features and processed through unification \cite{kay-1984-functional-unification-grammar}.
    
    \begin{itemize}
        \item Practically it is an attribute-value structure such as {x=1.2, y=3.2, z=4.67} for GPS
        \item \textbf{Trace} is the name used from here after becaus eit is shorter and it is before I knew ``Feature structure`` is also a linguistic term similar to attribute-value structures in computer science
    \end{itemize}

\end{frame}
