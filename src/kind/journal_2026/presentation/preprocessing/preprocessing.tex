\begin{frame}{Preprocessing}
    Both during the novelty detection or building the predictive model the sensory data should go through the following steps but here IMU data is used directly instead of calculating manually the derivitive
    \begin{itemize}
        \item Normalization
        \item Fusing different modalities using autoencoders
        \item clustering db scan of each sensors to find which sensor has one to many relationshipwith others (for one LIDAR there might be many GPS and IMU)
        \item Deducting the LIDAR 
    \end{itemize}

\end{frame}
